# Superstore-sales-dashboard
Interactive Excel dashboard analyzing $2.3M in sales data across 4 regions, 3 categories, and 3 customer segments using Microsoft Excel.

## Key Insights
- West region leads with highest sales ($725K)
- Technology is the most profitable category
- Consumer segment generates 47% of total revenue
- Sales show a consistent upward trend from 2014-2017
- Forecasted next 3 months' sales using trend analysis

## Tools Used
Microsoft Excel
Pivot Tables
Charts (Bar, Pie, Donut, Line)
Slicers
Forecasting (FORECAST function)

## Dashboard Features
KPI Cards (Total Sales, Profit, Orders, Discount)
Region-wise sales breakdown
Category-wise profit analysis
Monthly sales trend (2014-2017)
Segment wise revenue distribution
3 month sales forecast
